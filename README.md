Celem opracowania było przeprowadzenie kompleksowej analizy ekonomicznej i kosztowej trzech
wybranych systemów instalacji centralnego ogrzewania dedykowanych dla domu jednorodzinnego.
Analiza obejmuje zarówno koszty inwestycyjne, jak i eksploatacyjne, a także ocenę efektywności
ekonomicznej każdego z rozpatrywanych rozwiązań w kontekście długoterminowej opłacalności.
